# Map Art Direction

`map-art-director` 관점에서 정리한 적벽 전장 지도 교체 방향.

## 현재 문제

- 캔버스 렌더가 프로토타입 다이어그램처럼 보인다.
- 영토 면, 도로, 전선의 시각 언어가 모두 비슷해 전략 가독성이 약하다.
- 시작 전 화면 장식 지도와 전장 베이스맵의 톤이 뒤섞여 있다.

## 목표 구조

- 베이스맵: SVG 또는 이미지 자산
- 오버레이: 캔버스 상호작용

즉,
- 지형, 강줄기, 도시 기호, 지역 분위기: 아트 자산
- 선택, 전선 하이라이트, 이동 펄스, 클릭 히트테스트: 코드

## 시각 위계

1. 강과 지형 경계
2. 도시 기호
3. 세력 권역
4. 전선/연결선
5. 선택/이벤트 반응

## 금지

- stroke 두께와 blur만으로 맵 퀄리티를 해결하려는 접근
- 시작 전 화면 장식을 전장 퀄리티 개선으로 오인하는 접근
- 지형과 UI 톤이 서로 경쟁하는 접근

## 다음 구현 우선순위

1. `red-cliffs-base.svg`를 실제 전략 지도처럼 재디자인

3. 강/수역을 명확한 레이어로 분리
4. 전선은 베이스맵이 아니라 오버레이에서만 강조
5. 전장과 시작 화면의 장식 언어 분리
