## 삼국지 8 리메이크 (2024) — 장수 개인 플레이

삼국지8은 시리즈 중 유일하게 **장수 개인 시점 RPG**. 침착맨 부사인전의 기반.

### 행동력 시스템
- 매 턴(평정 후 도시행동) 행동력을 소비해서 활동
- 자택, 궁성, 주점, 시장 등 장소별 활동
- **핵심**: 한정된 행동력으로 뭘 할지 선택 → 전략적 깊이

### 신분 시스템
| 신분 | 할 수 있는 것 | 제약 |
|---|---|---|
| 방랑군/재야 | 전국 이동, 자유 행동 | 군사력 없음 |
| 일반 장수 | 내정 참여, 교류 | 인접 도시만 이동 |
| 태수 | 도시 관리, 소속 무장 관리 | 담당 도시만 |
| 군사/도독 | 전략 제안, 외교 | 군주 허가 필요 |
| 군주 | 모든 권한 | 책임도 큼 |

→ **추상화 포인트**: 신분에 따른 권한/시야/행동범위 차등. 승진 욕구가 동기부여.

### 교류 시스템
- 담화(20), 기증(20), 등용(20), 대련(20), 토론(20) → 친밀도 증가
- 연회: 최대 10명, 월 1회, 행동력 50
- **핵심**: 인간관계 = 자원. 친밀도 높은 장수는 전투/내정에서 지원해줌

### 단련/성장 시스템
- 행동력 60 → 능력치 경험치 습득
- 소양 소비 → 전법/특기 학습
- **핵심**: 캐릭터 빌드. 어떤 능력치/특기에 투자할지 선택

### 충성도/배신 메카닉
- 친밀도 (개인) vs 충성도 (세력) 분리
- 반란 시: 혈연 관계 → 혐오, 의형제/부부 관계 해제, 악명 +300
- **핵심**: 배신의 대가가 크다 → 드라마틱하지만 리스크도 큼

### 내정 5가지
| 활동 | 능력치 | 효과 |
|---|---|---|
| 개간 | 정치 | 군량 수입 ↑, 병사 상한 ↑ |
| 상업 | 정치 | 금 수입 ↑ |
| 기술 | 지력 | 기술력 → 기술서 → 병종 해금 |
| 보수 | 통솔 | 성벽/방어시설 내구 ↑ |
| 치안 | 무력 | 치안 → 반란 방지 |

### 외교 시스템 (군주/도독 전용)
- 친선: 금/군량/명품 → 우호도 ↑
- 동맹: 12~60개월, 원군 요청 가능
- 권고: 항복 유도
- 연합: 대상 세력 지정 → 다수 연합
- 계략: 이간/파괴/선동/내통/매복 (금 200 + 전략 포인트 1)

### 전투/병종 시스템
- 기술서 → 병종 트리 해금 (경보→중보→정예→만병)
- 보병/기병/궁병 3계열 + 병기(충차, 투석 등)
- **핵심**: 기술 개발 → 병종 해금 → 전투력 연쇄. 내정이 전투로 직결

### 연의전 (이벤트 시스템)
- 6가지 분류: 역사/교류/의뢰/목표/지도/종극
- 발생 조건 + 기한 + 플레이어 선택으로 분기
- **핵심**: 이게 우리 게임의 AI 확장 대상. 500개 → 50,000개

## 삼국지 14 (2020) — 군주 영토 전략

삼국지14는 **군주 시점 대전략**. 침착맨 적벽 망한 유비전의 기반.

### 영토 시스템
- 도시 46개 + 중요 지점 340개 = 토지 칠하기 (진삼국무쌍 + 전략)
- 토지를 점령하면 자동으로 병참선 형성
- **핵심**: "그림 그리듯" 영토 확장. 시각적으로 세력 팽창이 보임

### 군주 플레이
- 삼국지 9 계열 — 군주만 조작, 장수에게 지시
- 장수의 능력/성격에 따라 지시 이행도가 다름
- **핵심**: 인재 관리 = 전략. 좋은 장수를 어디에 배치하느냐

### IF 시나리오
- "적벽에서 조조가 이겼다면?" 등 대체역사 시나리오 지원
- **핵심**: 우리 게임의 AI 이벤트 생성과 직결. 역사의 분기점마다 "만약?"

## 추상화: 테마 독립적 메카닉

코에이 삼국지에서 "삼국지"를 빼면 남는 것:

### 1. 턴제 자원 관리
- 행동력 (시간 자원) → 무엇에 투자할 것인가
- 금/군량/병사 (물질 자원) → 내정/외교/전투 밸런스

### 2. 관계 네트워크
- 개인 친밀도 + 조직 충성도 이중 구조
- 관계의 종류: 군신, 의형제, 혼인, 원수, 사제
- 관계가 이벤트 트리거 + 전투 보너스 + 배신 확률에 영향

### 3. 신분/권한 계층
- 시작은 아무것도 아닌 존재 → 승진/독립 → 천하통일
- 신분마다 할 수 있는 것이 다름 → 성장 동기

### 4. 이벤트 드리븐 서사
- 조건 충족 → 이벤트 발화 → 선택지 → 결과
- 역사적 이벤트 + 랜덤 이벤트 + 캐릭터 관계 이벤트

### 5. 전투 = 준비의 결산
- 내정(기술 개발)이 전투력으로 직결
- 장수 능력 + 병종 + 진형 + 지형 + 보급
- 일기토 (1v1) = 드라마틱 순간

### 6. AI 세력 (비플레이어)
- 다른 세력도 같은 규칙으로 행동
- 외교/전쟁/내정을 자율적으로 수행
- 플레이어가 세계의 중심이 아님
