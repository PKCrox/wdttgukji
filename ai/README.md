# AI Workspace

이 디렉터리는 Claude 전용 서브에이전트 메모를 그대로 복사한 곳이 아니라, `wdttgukji`에서 Codex가 바로 사용할 수 있게 재구성한 운영 명세 레이어다.

## 구성

- `agents/`: 역할별 담당 범위, 입력, 출력, 판단 기준
- `workflows/`: 여러 에이전트가 연결되는 반복 루프

## 사용 원칙

- README의 비전이 맞는지 확인하고, 실제 코드는 `scripts/`, `engine/`, `public/`, `data/`에서 검증한다.
- 작업이 하나의 역할을 넘어서면 `pipeline-architect`를 기준으로 전체 입출력 계약부터 맞춘다.
- 삼국지 관련 지식이 필요하다고 해서 모든 것을 `historian` 류 설명으로 처리하지 않는다. 이 프로젝트의 핵심은 생산 파이프라인과 실험 루프다.
