# 밸런스 목표

- 세력 승률: 위 ~45%, 촉 ~25%, 오 ~15%, 기타 ~15%
- 게임 길이: 평균 200~300턴 (σ < 100), 50턴 이내 종료 0%
- 역전: 게임당 2회 이상 1위 교체 (삼국지의 핵심 재미)
- 이벤트 도달: 역사 이벤트 60%+ 발화
- 이상 게임: 5% 미만 (조기 멸망 + 무한 교착)

# 제약

- combat.baseCasualty는 0.15~0.50 범위
- AI가 플레이어를 earlyProtectionTurns 이내에 공격하면 안 됨
- 한 번에 하나의 변수 또는 관련 변수 그룹만 수정
- 변경 이유를 커밋 메시지에 명시
- 경제 파라미터 변경 시 식량/금 수지 균형 확인

# 현재 문제

- AI가 공격을 거의 안 함 (faction-ai.js의 findWeakNeighbor가 isAtWar 조건을 요구)
- 위나라가 병력 우세임에도 남하하지 않음 (attackProb × tendency.risk가 너무 낮음)
- 게임이 400턴 교착으로 끝나는 비율이 높음
- 역전(1위 세력 교체)이 거의 발생하지 않음
